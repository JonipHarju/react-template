## Getting Started

1. **Clone this repository and navigate into the folder:**

   ```bash
   git clone https://github.com/JonipHarju/react-template.git
   cd react-template
   ```

2. **Install dependencies:**
   ```
   npm install
   ```
3. **Do the exercises in "---" file inside the "src" folder.**

4. **To run the tests:**
   ```
   npm test
   ```
